Pro photographers can pick how the location list renders on the client's Pick page. There are six templates to choose from. They're all styled with your colors / font / logo from Branding — the difference is layout density and rhythm.

## The six templates

**Editorial** — long-form story per location. Big photo, headline-style location name, paragraph copy. The most "magazine feature article" feel. Good for a curated 4-6 location wedding guide where each spot deserves its own moment.

**Card** — the default. Tall photo per location with name + meta below. Clean, content-focused. Works for any guide size.

**Grid** — two-column tiled gallery. Fits more on screen. Good for portfolio guides with 15+ locations where you want the client to scan visually before tapping in.

**Magazine** — first card spans both columns as a hero, the rest fill a 2-up grid below. Reads like the cover of a print magazine. Good when you have one obvious flagship spot and want to lead with it.

**Compact list** — small thumbnail on the left, info + button on the right. Most info-dense. Good for guides where the photographer wants the client to focus on practical info (permit, parking) rather than scrollable photos.

**Minimal** — text-forward rows with a tiny thumbnail and a link. Most stripped-down. Good if you want the photo experience to live entirely in the detail panel rather than the index — clients get a list, tap a row, see the location's full photo gallery.

## Where to set it

**Profile → Branding tab → Layout templates section.** This is your default; every Location Guide uses it unless you override.

**Per-guide override**: in the guide editor (New Location Guide / Edit Location Guide), the **Location Guide template** section near the bottom lets you pick a different template for just that guide. Useful when you want wedding clients to see Editorial and engagement clients to see Magazine.

## Tweaking the look

The same Branding tab lets you customize the colors, font, and header for whichever template is active. The live preview at the top of the page repaints in real time as you tweak, so you can see what the client will see without leaving the page. The **👁 Full preview** button next to the live preview opens the actual Pick page in a desktop / mobile-toggleable iframe with your real portfolio.

## Plan tier

Layout templates and color / font customization are **Pro-only**. Free / Starter photographers see all guides in the default Card template with the LocateShoot brand colors, with no override option.
